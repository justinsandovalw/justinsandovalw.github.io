# Portfolio

Personal portfolio site focused on cybersecurity transition work, field networking experience, technical writeups, and resume access.

## Live Site

- [https://justinsandovalw.github.io](https://justinsandovalw.github.io)

## What Is Included

- `index.html` - Main landing page with:
  - Hero section and cybersecurity profile
  - Credentials and current certification progress
  - Project highlights
  - Expandable technical writeups
- `portfolio.css` - Shared styling and responsive layout for the portfolio page
- `resume.html` - Resume preview page with download options for both PDF and DOCX
- `resume.pdf` - Primary downloadable resume format
- `resume.docx` - Alternate downloadable resume format
